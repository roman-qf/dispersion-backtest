# Dispersion Trading Strategy Backtester

## Описание проекта
Этот проект представляет собой инструмент для бэктестинга (тестирования на исторических данных) дисперсионной торговой стратегии.

Алгоритм анализирует корзину из высоколиквидных акций (Blue Chips) и моделирует стратегию Relative Value, основанную на разнице доходностей между "ядром" портфеля и отстающими активами.

## Бизнес-контекст (Business Case)
Проект был разработан для помощи инвестиционным консультантам в продаже структурных продуктов (нот).

**Проблема:**
Инвестиционным консультантам требовался инструмент для быстрой демонстрации исторической эффективности ноты потенциальным клиентам. Ручной расчет доходности (бэктест) корзины активов и сравнение её с бенчмарком занимал много времени и имел риск человеческой ошибки.

**Решение:**
Данный скрипт автоматизирует процесс моделирования стратегии "Dispersion / Relative Value". Он позволяет:
1. Загрузить реальные рыночные данные за 10 лет.
2. Смоделировать механику выплат по структурной ноте (покупка корзины vs продажа аутсайдеров).
3. Сгенерировать понятный отчет (CAGR, вероятность получения дохода), который можно использовать в презентации для клиента.

**Результат:** Скрипт сокращает время подготовки аналитики с нескольких часов до секунд.

## Основная логика стратегии
Скрипт рассчитывает спред между:
1. Средней доходностью всей корзины активов (Benchmark).
2. Средней доходностью N худших активов (Worst Performers).

Цель — оценить устойчивость и профит-фактор такой дисперсии на длительном горизонте.

## Функциональность
* **Автоматическая загрузка данных:** Использование API `yfinance` для получения исторических цен (Adj Close).
* **Модульная архитектура:** Разделение логики на загрузку данных, расчет альфы, движок бэктеста и отчетность.
* **Аналитика:** Расчет CAGR (Compound Annual Growth Rate) и классификация сделок по доходности.
* **Визуализация:** Построение графиков распределения прибыли.

## Структура проекта
* `main.py` — Точка входа. Конфигурация параметров (тикеры, горизонт инвестирования) и запуск.
* `strategy.py` — Логика расчета торгового сигнала (дисперсии).
* `backtest.py` — Движок симуляции сделок скользящим окном.
* `data.py` — Модуль работы с API Yahoo Finance.
* `report.py` — Генерация метрик и графиков.
